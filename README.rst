Scheduler-cli tool
================================
Intro
-----
Scheduler for Python

Installation
-----------------

::

	pip install scheduler-cli

Usage
-----------------

1. Add schedule
-----------

::

	add {due} {content} in {category}

Example

::

	add 3/2 hit hoesung in school
